### 2. Router del servidor actualizado
Archivo: `edu-conect-rural-server/src/main.rs` (línea 109)

```rust
.nest_service("/modulos", ServeDir::new("../edu-conect-rural-dashboard/modulos")
    .append_index_html_on_directories(true))
```

Rutas activas:
- `/` → Landing page (HTML estático en `static/index.html`)
- `/app` → Redirect 302 a `/app/`
- `/app/` → Dashboard Next.js (`frontend_path = ../edu-conect-rural-dashboard/out/`)
- `/modulos/` → Catálogo + módulos (directorio independiente)
- `/api/*` → API REST (cursos, progreso, búsqueda)
- `/admin/*` → Panel admin
- `/zim/` → Wikipedia offline

## Lección aprendida

**Los módulos NO deben vivir dentro de `out/`** porque `npm run build` limpia ese directorio. Ubicación correcta: `edu-conect-rural-dashboard/modulos/` servida por ruta dedicada en el router Rust.
