### 3. Script de inyección del menú
Tras `npm run build`, el script se pierde (se regenera `out/`). Se reinyectó con Python antes de `</body></html>`. 
Procedimiento para futuros rebuilds:
```bash
cd edu-conect-rural-dashboard && npm run build
python3 -c "
p='out/index.html'
c=open(p).read()
s='<script>(function(){if(window.__edumod)return;...})();</script>'
open(p,'w').write(c.replace('</body></html>',s+'</body></html>'))
"
```
