## 7. Corrección de Bugs de Infraestructura

### Problema: Landing page mostraba HTML con números de línea
El archivo `static/index.html` se corrompió con números de línea (formato `1|contenido`) debido a patches previos. Se limpió con regex y se recompiló el binario Rust (`cargo clean + cargo build --release`).

### Problema: Servidor Rust no arrancaba
La base de datos SQLite quedaba con archivos WAL/SHM corruptos tras matar el proceso. Solución: eliminar `educonect.db-shm` y `educonect.db-wal` para forzar checkpoint.
