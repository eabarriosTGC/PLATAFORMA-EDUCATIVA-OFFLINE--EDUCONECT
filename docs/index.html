EduConect Rural

Plataforma educativa offline para escuelas rurales 🌵

La Guajira, Colombia — Comunidad Wayuu

🎯¿Qué es?

Un servidor educativo portátil que convierte una Raspberry Pi en un punto de acceso educativo offline. Estudiantes se conectan con sus celulares y acceden a Wikipedia, videos, módulos interactivos, diccionario wayuunaiki y más — sin internet.

🐙 Ver en GitHub 🐳 Probar con Docker

🐳Probar con Docker

En un solo comando:

git clone https://github.com/eabarriosTGC/PLATAFORMA-EDUCATIVA-OFFLINE--EDUCONECT
cd PLATAFORMA-EDUCATIVA-OFFLINE--EDUCONECT
docker compose up -d
# Abrir http://localhost:8080

O construir manualmente:

docker build -t educonect-rural .
docker run -p 8080:8080 educonect-rural

🛠️Stack

Rust + Axum SQLite / FTS5 Next.js Vanilla HTML/CSS/JS zim-rs ffmpeg yt-dlp espeak-ng PWA Docker
